# Future Features

Planned improvements for readrun.

## Implemented

- **[Static site deployment](static-site-deployment.md)** — GitHub Pages, Vercel, Netlify builds via TUI
- **[Link navigation](link-navigation.md)** — markdown links rewritten to rendered paths
- **Resource browser** — sidebar tabs for content, images, files, scripts
- **Context menu** — right-click for Search and Settings
- **In-page search** — `/` to search with highlighted matches and prev/next navigation
- **Table of contents** — auto-generated TOC sidebar from headings
- **Themes** — 8 built-in themes with theme picker overlay
- **Keyboard shortcuts** — fully configurable via `~/.config/readrun/settings.toml`
- **File downloads** — files generated by Pyodide scripts are detected and offered as downloads
- **Auto package installation** — imports are detected and packages installed via micropip automatically
- **Package preloading** — page code blocks are scanned on load and packages install in the background
- **Matplotlib inline rendering** — `plt.show()` renders plots directly in the output area

## Remaining

- **Client-side routing** — intercept internal links for instant navigation without page reloads
- **Frontmatter dependencies** — declare Python dependencies in page YAML frontmatter
- **Client-side file uploads** — let readers upload files into Pyodide's virtual filesystem for processing
- **Broken link detection** — warn during build if markdown links point to missing files

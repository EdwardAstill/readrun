# Lecture 1: Python Basics

## Variables and Types

:::python
x = 42
name = "Alice"
pi = 3.14159

print(f"x = {x} (type: {type(x).__name__})")
print(f"name = {name} (type: {type(name).__name__})")
print(f"pi = {pi} (type: {type(pi).__name__})")
:::

## Lists

:::python
fruits = ["apple", "banana", "cherry"]
for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")
:::

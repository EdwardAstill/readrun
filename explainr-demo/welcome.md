# Welcome

This is a demo site built with **explainr**.

Browse the sidebar to see different pages. Code blocks marked with `:::` are executable (coming soon).

## How it works

Write Markdown files, run `explainr`, and you get an interactive website.

```
my-project/
  welcome.md
  notes/
    lecture-1.md
    lecture-2.md
  tutorials/
    intro.md
```
